# java-plugins

分享一些自己在日常中封装的一些组件、spring-boot-starter、工具包等

## 快速开始

### 目录结构

```lua
java-plugins
├── java-plugins-pom -- 公共版本控制
├── spring-boot-tools -- spring boot 常用的一些工具类
└── spring-redisson-queue-starter -- 基于redisson实现的队列组件
```
